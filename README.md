# NJUPTThesis
Bachelor Thesis Template of NJUPT

# 使用说明
1.适用 TeX Live 2015/ 2016 软件环境. 

2.使用 XeLaTeX 编译.

3.主文档是 Bachelor-template.tex .

4.图片请放在 figures 文件夹.

5.includefile 存放的是中英文摘要、致谢等等正文以外的部分。
  请直接在这些文件中填写相关内容,不要改变文件的位置.

6.更多使用说明请参看源文件 Bachelor-template.tex 里的注释说明.
